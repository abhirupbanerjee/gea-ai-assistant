### Key Features - Standalone Mode

**Header:**
- GEA icon (36x36px)
- "GEA Bot" title (text-lg/xl)
- Connection status badge (green, animated pulse)
- Context summary (if connected to portal page)
- White background with shadow

**Welcome Screen:**
- Large bot icon with gradient background (green to emerald)
- Professional welcome title
- Detailed EA Portal-focused description
- Three prominent quick-start buttons:
  1. Blue gradient: Explain Grenada EA Framework
  2. Emerald gradient: What is the role of DTA?
  3. Purple gradient: Provide an overview of EA Portal
- All buttons auto-submit on click
- Active scale effect on button press
- "Or type your own question" hint

**Input Area:**
- 3-row textarea (text-base, 16px)
- Standard padding (px-4 py-3)
- Copy, Clear, and Send buttons
- Full button sizes with tooltips

### Key Features - Embedded Mode

**No Internal Header:**
- Header is provided by parent portal
- Parent shows: "Grenada AI Assistant" + context + close button
- GEA Bot content starts immediately with chat container

**Chat Container:**
- Reduced padding (p-2.5 vs p-3)
- Compact spacing between messages (gap-3 vs gap-4)
- Thinner scrollbar (8px custom styled)
- Messages have blue left border accent

**Welcome Screen (if shown):**
- Same bot icon but may be smaller
- Generic portal navigation message
- Three general-purpose quick actions:
  1. How do I submit feedback?
  2. How do I file a grievance?
  3. What services are available?

**Input Area:**
- 2-row textarea (text-sm, 14px)
- Compact padding (px-3 py-2.5)
- Smaller buttons (p-2, 16px icons)
- Reduced gaps (gap-1.5 vs gap-2)

- Bot detects current page/route from parent portal
- Shows page-specific welcome message
- Displays context summary in parent header

- Dynamically generated based on current page
- Generic but contextual prompts:
  1. What can I do on this page?
  2. How do I complete the main task here?
  3. Show me step-by-step instructions

- `/` - Portal home
- `/feedback` - Feedback submission
- `/grievance` - Grievance filing
- `/admin/analytics` - Analytics dashboard
- `/admin/grievances` - Grievance management

## 6. Design Specifications

### Colors

**Primary Blue:**
- Blue-50: `#eff6ff` (light backgrounds)
- Blue-100: `#dbeafe` (borders)
- Blue-400: `#60a5fa` (accent borders)
- Blue-500: `#3b82f6` (primary)
- Blue-600: `#2563eb` (buttons)
- Blue-700: `#1d4ed8` (hover)

**Emerald:**
- Emerald-50: `#ecfdf5` (light backgrounds)
- Emerald-100: `#d1fae5` (borders)
- Emerald-500: `#10b981` (icons)
- Emerald-600: `#059669` (gradients)

**Purple:**
- Purple-50: `#faf5ff` (light backgrounds)
- Purple-100: `#f3e8ff` (borders)

**Grays:**
- Gray-100: `#f3f4f6`
- Gray-200: `#e5e7eb`
- Gray-500: `#6b7280`
- Gray-600: `#4b5563`
- Gray-800: `#1f2937`
- Gray-900: `#111827`

### Typography

**Standalone Mode:**
- Title: `text-2xl sm:text-3xl` (24-30px)
- Header: `text-lg sm:text-xl` (18-20px)
- Body: `text-base` (16px)
- Description: `text-lg` (18px)
- Small text: `text-xs` (12px)

**Embedded Mode:**
- Title: `text-base` (16px)
- Body: `text-sm` (14px)
- Small text: `text-xs` (12px)

### Spacing

**Standalone:**
- Container: `p-4` (16px)
- Messages: `gap-4` (16px)
- Input: `px-4 py-3`
- Buttons: `p-3`

**Embedded:**
- Container: `p-2.5 to p-3` (10-12px)
- Messages: `gap-3` (12px)
- Input: `px-3 py-2.5`
- Buttons: `p-2`

### Shadows

- Header: `shadow-md`
- Chat container: `shadow` (standalone), `shadow-sm` (embedded)
- Buttons: `shadow-sm hover:shadow-md`
- Messages (embedded): `shadow-sm hover:shadow-md`

### Border Radius

- Containers: `rounded-xl` (12px)
- Buttons: `rounded-lg` (8px)
- Messages: `rounded-lg` (8px)
- Input: `rounded-lg` (8px)

### Transitions

- All interactive elements: `transition-all duration-200`
- Buttons: `active:scale-[0.98]`
- Hover states: Smooth color and shadow transitions

## 7. Accessibility Features

### ARIA Labels
- All icon buttons have `aria-label` attributes
- Send button: "Send message"
- Copy button: "Copy chat"
- Clear button: "Clear chat"
- Close button (embedded): "Close assistant"

### Keyboard Navigation
- Enter to send message
- Shift+Enter for new line
- Tab navigation through all interactive elements
- Focus indicators on all buttons and inputs

### Visual Feedback
- Hover states on all interactive elements
- Active states with scale effects
- Disabled states with opacity changes
- Loading indicators (spinning hourglass)
- Tooltips on hover (bottom-positioned)

### Screen Reader Support
- Semantic HTML structure
- Proper heading hierarchy
- Descriptive button text
- Alternative text for images

const { isEmbedded, hasContext } = usePageContext();
