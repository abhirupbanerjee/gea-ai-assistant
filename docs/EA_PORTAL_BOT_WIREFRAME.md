## Component Details

### Header Section
- **Logo**: 48x48px (sm:), rounded
- **Title**: "EA Portal Bot" - text-lg sm:text-xl, font-bold
- **Context Badge**: Green pill with pulse animation (when embedded)
  - Shows "Connected" with animated dot
  - Only visible when page context is available

### Message Bubbles
- **Avatar**: 32x32px circular
  - User: Blue solid (bg-blue-500)
  - Bot: Green gradient (from-green-500 to-emerald-600)
- **Name & Timestamp**: Flex row, space-between
- **Content**:
  - User: Blue background (bg-blue-50)
  - Bot: White with border and shadow
- **Copy Button**: Appears on hover, top-right of message

### Input Section
- **Textarea**: 3 rows, full width, border-2, rounded-lg, resize-none
  - Supports Shift+Enter for new line
  - Enter to send message
- **Action Buttons** (single row, 25%-25%-50% width distribution):
  - **Copy** (25%): Icon-only, white background, left side
  - **Clear** (25%): Icon-only, white background with red hover
  - **Send** (50%): Icon-only, blue background, largest button
  - All buttons have tooltips on hover
  - All buttons are icon-only with centered icons

### Color Scheme
- **Primary**: Blue (#2563eb)
- **Bot Brand**: Green-Emerald gradient
- **Borders**: Gray-300 (#d1d5db)
- **Hover**: Gray-100 (#f3f4f6)
- **Text**: Gray-800/900 (#1f2937)

### Welcome Screen Prompts
**Context-Based (Embedded in Portal)**:
- **Prompt 1**: "What can I do on this page?" - Blue gradient
- **Prompt 2**: "How do I complete the main task here?" - Emerald gradient
- **Prompt 3**: "Show me step-by-step instructions" - Purple gradient

**Generic (Standalone)**:
- **Prompt 1**: "How do I submit feedback?" - Blue gradient
- **Prompt 2**: "How do I file a grievance?" - Emerald gradient
- **Prompt 3**: "What services are available?" - Purple gradient

The bot automatically switches between context-based and generic prompts based on whether it's embedded in the portal or used standalone.

### Responsive Breakpoints
- **Mobile**: Full-width layout, stacked elements
- **Textarea**: Always full width across all breakpoints
- **Buttons**: Always in horizontal row with 25%-25%-50% distribution
- **Prompts**: Full-width buttons with gradient backgrounds

## User Interaction Flow

1. **Click Quick Action Prompt** â†’ Fills textarea and hides welcome screen
2. **Type in Textarea** â†’ Enter to send, Shift+Enter for new line
3. **Hover Action Buttons** â†’ Tooltip appears above button
4. **Click Send** â†’ Message appears with timestamp, bot responds
5. **Hover Message** â†’ Copy icon fades in
6. **Click Copy (message)** â†’ Checkmark appears for 2 seconds
7. **Click Copy Chat** â†’ Entire conversation copied to clipboard
8. **Click Clear Chat** â†’ All messages cleared, welcome screen returns

## Key Improvements vs Original Version

| Feature | Original | Current |
|---------|----------|---------|
| Input Type | Single-line input | Multi-line textarea (3 rows) |
| Input Shortcuts | Enter only | Enter to send, Shift+Enter for newline |
| Action Buttons Layout | 2 rows (input+send, copy+clear) | 1 row (copy, clear, send) |

| Branding | "Grenada AI Assistant" | "EA Portal Bot" |

| Welcome Prompts | Static EA/DTA topics | Dynamic context-based prompts |
| Header Controls | Size toggle button | Context indicator only |
| Chat Width | Resizable (4 modes) | Fixed 896px (max-w-4xl) |
